üöÄ Quick Start

Getting Started in 5 Minutes

  1. Clone Repository: git clone https://github.com/Anteros79/KiroMLE-STAR.git
  2. Run Setup: setup.bat (Windows) or ./setup.sh (Linux/Mac)
  3. Start Services: run-all.bat or run each service separately
  4. Access UI: Open http://localhost:3000 in your browser
  5. Upload Dataset: Use the UI to upload your competition dataset

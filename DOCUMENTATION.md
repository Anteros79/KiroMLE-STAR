## ğŸ–¥ï¸ Windows Batch Files

For Windows users, convenient batch files are provided:

- **setup.bat** - Initial environment setup and configuration
- **run-backend.bat** - Start the FastAPI backend server
- **run-frontend.bat** - Start the Next.js frontend server
- **run-all.bat** - Start all services in separate windows
- **run-tests.bat** - Run the complete test suite

### Usage

```bash
# Initial setup
setup.bat

# Start all services
run-all.bat

# Or start individual services
run-backend.bat
run-frontend.bat

# Run tests
run-tests.bat
```
